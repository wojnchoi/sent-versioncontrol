# SENT - simple version control system

SENT is a version control system.   
It allows to keep and record the changes of the file.
   
#### install
```
// using this source code
% make install
```
#### cmds
```
mail   -   start sent database(inbox) 
send   -   send files to a repository     
back   -   back to a previous(sha1) repository
config -   display user
```
